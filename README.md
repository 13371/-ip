# 砚白配置IP

在 Windows 10 及以上系统托盘一键切换**手动 IP/DNS** 与 **自动 DHCP**，支持多模板、断网自动切回 DHCP 并关闭代理。

## 功能

- **一键切换旁路由 IP**：从托盘选择模板即可应用该模板的静态 IP、网关、DNS。
- **一键切换自动 DHCP**：从托盘选择「切换到自动 DHCP」即可。
- **多模板**：支持保存多个 IP 模板，适配不同网络环境；支持新增、编辑、删除。
- **网络监测（始终开启）**：检测到断网时自动切换为自动 DHCP，并关闭「网络和 Internet → 代理」中的代理。
- **开机启动**：可选开机自动运行（托盘菜单「开机启动」勾选）；使用计划任务以**管理员权限**启动，重启后无需再手动重新运行。
- **轻量**：仅依赖 Python 与少量库，常驻托盘，占用资源小。

## 环境与依赖

- Windows 10 及以上
- Python 3.8+

安装依赖（任选其一）：

- 双击运行 **`安装依赖.bat`**（推荐）
- 或在终端中执行（若直接输 `pip` 报错，请用下面两种之一）：
  ```bash
  python -m pip install -r requirements.txt
  ```
  若系统有 Python 启动器，也可用：
  ```bash
  py -m pip install -r requirements.txt
  ```

## 使用与权限

- **必须以管理员身份运行**，否则无法通过 `netsh` 修改网卡 IP/DNS。
- 推荐方式：右键「以管理员运行」本目录下的 `以管理员运行.bat`；或右键「以管理员身份运行」命令提示符后执行：
  ```bash
  python "d:\配置ip\main.py"
  ```
- 首次使用请先在托盘菜单中打开「管理模板」，添加至少一个 IP 模板（名称、IP、掩码、网关、DNS 等），之后即可在托盘菜单中一键应用或切换 DHCP。
- **开机启动**开启后，通过计划任务在用户登录时以**最高权限**运行，重启电脑后可直接使用（若系统开启 UAC，登录时可能出现一次「是否允许更改」的提示，点「是」即可）。

## 配置与数据

- 模板与「开机启动」选项保存在程序目录下：  
  `d:\配置ip\.yanbai_ip_config\config.json`  
  这样以管理员或普通用户运行时都使用同一份配置，避免「模板不存在」。
- 可直接编辑该 JSON 文件修改模板，或通过「管理模板」界面操作。

## 项目结构

- `main.py`：入口，系统托盘与菜单、网络监测入口。
- `config.py`：配置与模板的读写。
- `network_manager.py`：通过 netsh 设置静态 IP/DHCP、DNS。
- `proxy_manager.py`：关闭当前用户系统代理。
- `monitor.py`：后台网络监测，断网时触发切 DHCP + 关代理。
- `template_editor.py`：模板管理弹窗（新增/编辑/删除/应用）。
- `network_status_window.py`：网络状态详情窗口（状态、失败次数、触发次数等）。
- `autostart.py`：开机启动（计划任务，登录时以管理员运行）。
- `icon_gen.py`：生成托盘图标。

## 在其他电脑运行（无需安装 / 或需安装环境）

### 方式一：打包为 exe（推荐，其它电脑什么都不用装）

在本机（已安装 Python 的电脑）双击 **`build_exe.bat`**，会在 `dist` 目录生成 **`砚白配置IP.exe`**。

- 将 **`砚白配置IP.exe`** 复制到 U 盘或其它电脑即可。
- 在其它电脑上**无需安装 Python 或任何依赖**，直接运行 exe。
- 运行方式：右键「以管理员身份运行」该 exe（或双击时若弹出 UAC 提示，点「是」）。  
  exe 已带“请求管理员权限”的清单，首次运行通常会弹出 UAC。
- 配置和模板会保存在 **exe 所在目录**下的 `.yanbai_ip_config` 文件夹，便于随 exe 一起拷贝。

### 方式二：不打包，在其它电脑用 Python 运行（需安装环境）

若不想打包，在其它电脑需要：

1. 安装 **Python 3.8 或以上**（从 [python.org](https://www.python.org/downloads/) 下载，安装时勾选 “Add Python to PATH”）。
2. 将本项目的**整个文件夹**（包含所有 .py 和 requirements.txt）复制到该电脑。
3. 在该文件夹打开命令行，执行：
   ```bash
   python -m pip install -r requirements.txt
   ```
4. 以**管理员身份**运行：右键「以管理员身份运行」命令提示符，再执行：
   ```bash
   python "本机上的路径\main.py"
   ```
   或使用同目录下的「以管理员运行.bat」（需把批处理里的路径改成该电脑上的实际路径）。
